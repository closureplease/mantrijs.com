Mantri on the Command Line

NOTE: COMMAND LINE HAS NOT YET BEEN IMPLEMENTED! THE FOLLOWING IS JUST A DOODLE

Install mantri globaly and run it from the command line:

$ npm install -g mantri-cli

Next go to the root of your web project (what would be called your Document Root) and:

$ mantri init [your/bootstrap/file.js]

This will perform the following actions: 1. Create the mantriConf.json file. [The Configuration file][config-wiki] for mantri. 2. Create the mantri.js file. This file must only be included in your development environment. 3. Scan your folders and create the deps.js file which will contain information about your dependency tree.

That's it!

Read more about the command line options in [this wiki page][cli-wiki].
